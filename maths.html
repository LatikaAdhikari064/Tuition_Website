ALGEBRA

An algebraic expression is a combination of variables, numbers, and operations.

Example:

3x + 5y − 7

Here,

  • x and y are variables
  • 3 and 5 are coefficients
  • −7 is a constant

Types of Algebraic Expressions:

  • Monomial → Single term (e.g., 5x2)
  • Binomial → Two terms (e.g., x + 3)
  • Trinomial → Three terms (e.g., x2 − 4x + 7)
  • Polynomial → Multiple terms

2. Equations and Inequalities

Linear Equations

An equation where the highest power of the variable is 1.

Example: 2x + 3 = 7

Solution:

2x = 7 - 3

x = 4 / 2 = 2

Quadratic Equations

An equation of the form: ax2 + bx + c = 0

Example: x2 - 5x + 6 = 0

Factoring Method:

(x - 2)(x - 3) = 0

So, x = 2 or x = 3

Quadratic Formula:

x = (-b ± √(b2 - 4ac)) / 2a

Inequalities

Similar to equations, but involve <, >, ≤, ≥.

Example: 3x - 2 > 7

3x > 9

x > 3

3. Polynomials

A polynomial is an expression with multiple terms, where the exponents of the variables are whole numbers.

Example: 5x3 - 2x2 + 3x - 4

4. Functions and Graphs

A function is a relation where each input has exactly one output.

Example: f(x) = x2 + 2x - 3

5. Factorization

Factorization is breaking an expression into simpler terms (factors).

Example: x2 - 9 = (x - 3)(x + 3)

6. Algebraic Identities

(a + b)2 = a2 + 2ab + b2

(a - b)2 = a2 - 2ab + b2

a2 - b2 = (a - b)(a + b)

7. Logarithms (Advanced Algebra)
